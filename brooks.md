# Il "testo"

 è un insieme coerente di parole
 
che trasmette un messaggio, fondamentale nella comunicazione. Può essere scritto o orale, inclusi libri, leggi, articoli o produzioni artistiche. Sinonimi includono scritto, brano, passo, contenuto o opera. In cucina, il "testo" è anche una piastra in metallo per la cottura.
